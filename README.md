# TON Mini-Shop

A Telegram Mini App store on The Open Network (TON): connect wallet → buy a digital item → pay in TON → on-chain verification → bot delivers.

> Status: 🚧 in development. See [ROADMAP.md](./ROADMAP.md).

## Stack
- React + TypeScript + Vite + `@telegram-apps/sdk`
- `@tonconnect/ui-react` (TON Connect)
- Node + grammY (bot backend)
